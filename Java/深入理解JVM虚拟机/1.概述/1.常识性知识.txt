Sun官方所定义的Java技术体系包括以下几组部分：
	Java程序设计语言；
	各种硬件平台上的Java虚拟机；
	Class文件格式；
	Java API类库；
	来自商业机构和开源社区的第三方Java类库；
	

Java技术体系可以分得4个平台：
	Java Card：支持一些Java小程序（Applets）运行在小内存设备（如智能卡）上的平台；
	Java ME（Micro Edition）：支持Java程序运行在移动终端（手机，PAD）上的平台，对Java API有所精简，并加入了针对移动端的支持，这个版本以前称为J2ME；
	Java SE（Standa Edition）：支持面向桌面级应用（如windows下的应用程序）的Java平台，提供了完整的Java核心API，这个版本以前称为J2SE；
	Java EE（Enterprise Edition）:支持使用多层架构的企业应用，（如ERP，CRM应用）的Java平台，除了提供Java SE API外，还对其做了大量的扩充（一般以javax.*为包名）并提供了相关部署支持，这个版本以前称为J2EE；
